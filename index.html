01 · Pain Point

Overtourism is eroding Chiang Mai's heritage

Chiang Mai receives millions of tourists annually, yet no intelligent system integrates crowd data, visitor sentiment, and spatial intelligence to provide actionable guidance.

We built an Agentic RAG system that fuses GNSS trajectory analysis, YouTube sentiment mining, and LLM reasoning to answer overtourism-related questions with real spatial context.

02 · System Architecture

Four-step intelligent pipeline

From raw data to agentic answers — fully open-source and free

Step 1

YouTube Data API · 8 POI queries · English + location filtering · cardiffnlp XLM-RoBERTa sentiment model

sentiment_data.csv

Step 2

OSM GPS points · correlated random walk simulation · DBSCAN ε=0.002° · speed filter 150 km/h

hotspots.geojson

Step 3

Sentence Transformer 384-dim · FAISS IndexFlatIP cosine · 38 query eval · Hit Rate @5 = 78.9%

faiss.index

Application

Groq LLaMA 3.1 8B · ReAct agent loop · 4 tools: get_hotspot, get_sentiment, search_poi, rag_retrieve

streamlit run app.py
